# Technology Stack

## Core Engine
- **Godot 4.6 (Mono/C#):** The primary game engine and development environment.

## Programming Language
- **C#:** Targeting .NET 8.0, utilizing the standard Godot .NET SDK for game logic and scripting.

## Target Platforms
- **Windows:** Primary development and release target.
- **Android:** Secondary target (configured in project settings).

## Development Tools
- **Godot Editor:** For scene composition, level design, and asset management.
- **IDE:** Visual Studio or VS Code (implied for C# development).
- **System.Text.Json:** For level data serialization and deserialization.
- **NUnit:** For unit testing core utility logic.
